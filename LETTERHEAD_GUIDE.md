# B2P International — Letterhead Image Guidelines

## Optimal Image Specifications

### Header Image
- **Dimensions:** 794 x 150 pixels (or any 5.3:1 ratio)
- **Format:** JPG or PNG
- **Usage:** Stretches edge-to-edge across top of every page
- **Design Tip:** Include company logo, name, address, and contact details

### Footer Image
- **Dimensions:** 794 x 75 pixels (or any 10.5:1 ratio)
- **Format:** JPG or PNG
- **Usage:** Stretches edge-to-edge across bottom of every page
- **Design Tip:** Include copyright, social media icons, website, phone

### Company Logo
- **Dimensions:** 200 x 200 pixels (1:1 square ratio)
- **Format:** PNG with transparent background (recommended)
- **Usage:** Used in fallback header when no header image is provided
- **Design Tip:** Keep simple and clear, works best with transparent backgrounds

### Company Seal
- **Dimensions:** 200 x 200 pixels (1:1 square ratio)
- **Format:** PNG with transparent background (recommended)
- **Usage:** Appears on bottom-left of documents
- **Design Tip:** Circular or square official company seal/stamp

### Authorized Signature
- **Dimensions:** 400 x 120 pixels (or any 3:1 ratio)
- **Format:** PNG with transparent background (highly recommended)
- **Usage:** Appears on bottom-right of documents above "Authorized Signature" line
- **Design Tip:** Scan of actual signature, transparent PNG works best

## Image Preparation Tips

1. **Use High Resolution:** Minimum 72 DPI, 150 DPI preferred for print quality
2. **Transparent Backgrounds:** Use PNG with transparency for logo, seal, and signature
3. **Consistent Branding:** Use your brand colors and fonts
4. **Test Before Upload:** Preview images at actual size before uploading
5. **File Size:** Keep under 500KB per image for faster loading

## Color Modes
- **RGB** for digital/screen display (recommended)
- **CMYK** if you plan to print frequently

## Aspect Ratios Matter
The system will stretch images to fit, so maintaining proper aspect ratios prevents distortion:
- **Header:** Wide horizontal (5:1 or similar)
- **Footer:** Wide horizontal (10:1 or similar)  
- **Logo/Seal:** Square (1:1)
- **Signature:** Wide horizontal (3:1 or 4:1)

## Need Help?

If images appear stretched or misaligned:
1. Check the aspect ratio matches recommendations
2. Verify image dimensions (not file size)
3. Use transparent PNG for logo/seal/signature
4. Re-upload with correct dimensions

For best results, design your header and footer in image editing software (Canva, Photoshop, GIMP) at exact recommended dimensions before uploading.
