## 📋 Step-by-Step Creation Guide

### Step 1: Create Header (794 x 150px)

**Using Canva:**
1. Create custom size: 794 × 150 pixels
2. Set background: White
3. Add rectangle (bottom): 794 × 30px, color `#2C3E50`
4. Upload your logo (40×40px, top-left)
5. Add text: "B2P International" (24px, bold, red)
6. Add text: "Excellence in Business Solutions" (11px, gray)
7. Add text: Address in blue bar (9px, white)
8. Add diagonal shape (right side): Orange `#FF6633`
9. Download as PNG (794×150px)

**Using Photoshop/Photopea:**
1. New document: 794 × 150 pixels, 300 DPI
2. Create layers:
   - Background (white)
   - Bottom bar (blue, 30px height)
   - Logo placement
   - Company name text
   - Tagline text
   - Address text
   - Diagonal accent shape
3. Export as PNG

### Step 2: Create Footer (794 x 75px)

1. New document: 794 × 75 pixels
2. Add top bar: 25px, blue `#2C3E50`
3. Add white section: 50px, with gray top border
4. Add contact icons and text (centered)
5. Add social media text (bottom)
6. Add orange accent (small diagonal on right)
7. Download as PNG

### Step 3: Create Seal (200 x 200px)

1. New document: 200 × 200 pixels, transparent background
2. Draw circle: 200px diameter, red `#C84234` stroke (8px)
3. Add inner circle: 170px, white fill, orange border
4. Add curved text: "B2P INTERNATIONAL" (top curve)
5. Add center text: "B2P" (large, bold)
6. Add "INTERNATIONAL" below
7. Add "EST. 2015" or founding year
8. Add decorative stars
9. Download as PNG (transparent background)

### Step 4: Prepare Signature (400 x 120px)

**Option A — Scan:**
1. Sign on white paper with black/blue pen
2. Scan at 300 DPI
3. Open in editor
4. Remove background (make transparent)
5. Resize to max 400×120px
6. Save as PNG

**Option B — Digital:**
1. New document: 400 × 120 pixels, transparent
2. Choose cursive font (Allura, Great Vibes)
3. Type name, size 48px, color `#2C3E50`
4. Add slight italic/rotation
5. Export as PNG

## 💡 Design Tips

### Do's ✅
- ✅ Keep logo clearly visible (40×40px minimum)
- ✅ Use consistent brand colors throughout
- ✅ Ensure text is readable (9px minimum for print)
- ✅ Use high resolution (150 DPI minimum)
- ✅ Test on actual A4 paper before finalizing
- ✅ Use transparent PNG for seal and signature
- ✅ Maintain white space for breathing room
- ✅ Align elements to grid for clean look

### Don'ts ❌
- ❌ Don't use low-resolution images (pixelated)
- ❌ Don't use too many fonts (2-3 maximum)
- ❌ Don't make text too small (< 8px)
- ❌ Don't use busy backgrounds (keep clean)
- ❌ Don't stretch images (maintain aspect ratio)
- ❌ Don't use conflicting colors
- ❌ Don't overcrowd header/footer
- ❌ Don't forget to test printing

## 🖨️ Printing Tips

### For Best Print Results:
1. **Use CMYK color mode** if printing frequently
2. **Resolution:** 300 DPI for print-quality
3. **Bleed:** Add 3mm bleed if edge-to-edge printing
4. **Test print:** Always print a sample first
5. **Paper:** Use good quality paper (100 GSM minimum)
6. **Printer settings:** 
   - Color: "Best" or "High Quality"
   - Paper type: "Plain" or "Presentation"
   - Print as "Actual Size" (not scaled)
